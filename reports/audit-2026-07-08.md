## 🔴 Point en attente de vérification manuelle — mot de passe admin

**`seed.js` contient toujours `admin@immobf.africa / admin2026` en clair** (usage développement uniquement). Le mécanisme de reset fonctionne (#89). Il n'est pas possible de confirmer par code si le mot de passe de production a été changé.

**Action requise (une seule fois) :** tenter de se connecter sur [immoafrica.online/login](https://immoafrica.online/login) avec `admin@immobf.africa` / `admin2026`. Si la connexion réussit → changer immédiatement via `/admin/profile`. Si elle échoue → le mot de passe a bien été modifié, aucune action nécessaire.
